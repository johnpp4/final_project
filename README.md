# final_project
Inspired by the game "Wavelength", a popular mind-reading game between two people or large groups of friends. With this version I hope to create, you can utilize AI to help generate prompts for the player. The user can give prompts like "Give me an idea/topic that follows X theme and a word to go with the topic", this will help the user make a rating or "frequency" of that word and the other player will have to move the arrow to make a guess of the frequency rating. I feel like when playing Wavelength, me and my friends have a hard time thinking of a specific word and sometimes topics when playing the game so this interactive website helps solve that issue. The vision of this website is to be played on one device and both users/groups interact with it.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
